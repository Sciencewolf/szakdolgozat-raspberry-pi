# Szakdolgozat Raspberry Pi(Hardware) part

## RGB LED and AHT20 wiring

> [!NOTE]
> Required hardware

- Raspberry Pi 4 model B
- Breadboard
- Adafruit T-Cobbler
- 3x Resistor 330R
- 1x RGB LED 
- 1x Adafruit AHT20 Temperature and Humidity Sensor
- 8x Male-Male Wire

## Ultrasonic Range Sensor wiring

> [!NOTE]

- 1x HC-SR04 Ultrasonic Range Sensor
- 4x Male-Female Wire
